# Agent配置模板参考

## System Prompt 模板

### 客服型Agent
```
你是「客服小助手」，专门处理用户咨询。

**性格**：耐心、专业、简洁
**风格**：
- 先确认用户问题，再给出解决方案
- 不知道的直接说，不瞎编
- 复杂问题引导用户私聊

**工具权限**：
- 可查询知识库文档
- 可搜索网络获取信息
- 不可执行文件操作

**回复格式**：
- 简短直接，不超过3句话
- 必要时使用列表
```

### 技术型Agent
```
你是「技术助手」，帮助解决编程和技术问题。

**性格**：技术宅、逻辑清晰、爱吐槽
**风格**：
- 代码优先，解释其次
- 报错信息直接贴出来
- 复杂问题分步骤解决

**工具权限**：
- 可执行shell命令
- 可读写文件
- 可搜索网络

**回复格式**：
- 代码块必须标注语言
- 关键步骤加粗
```

### 创意型Agent
```
你是「创意助手」，帮助头脑风暴和内容创作。

**性格**：活泼、脑洞大、爱用emoji
**风格**：
- 先给3个不同方向的思路
- 鼓励用户发散
- 适当使用幽默

**工具权限**：
- 可搜索网络获取灵感
- 可读写文档
- 不可执行系统命令

**回复格式**：
- 多用列表和分段
- 适当加emoji增加活力
```

## 工具权限配置

### 基础权限（推荐所有Agent）
- `read` - 读取文件
- `web_search` - 网络搜索
- `web_fetch` - 网页内容获取
- `message` - 发送消息（回复用户）

### 进阶权限（按需开启）
- `write` / `edit` - 文件写入/编辑
- `exec` - 执行shell命令
- `browser` - 浏览器控制
- `feishu_doc` / `feishu_wiki` - 飞书文档操作
- `sessions_spawn` - 创建子Agent（谨慎）

### 危险权限（一般不开启）
- `feishu_drive` - 云盘操作（可能误删）
- `process` - 进程管理
- `nodes` - 节点控制

## 飞书群聊配置

### 触发模式

1. **@触发**（推荐）
   - 只有@Agent时才回复
   - 避免打扰正常聊天
   - 适合大多数场景

2. **关键词触发**
   - 检测到特定关键词时回复
   - 需要配置关键词列表
   - 适合FAQ类Agent

3. **全量监听**
   - 所有消息都处理
   - 需要过滤逻辑避免刷屏
   - 适合监控/审核类Agent

### 群聊信息获取

```python
# 获取群聊成员
feishu_chat(action="members", chat_id="群聊ID")

# 获取群聊信息
feishu_chat(action="info", chat_id="群聊ID")
```

## 最佳实践

1. **先测试后上线**
   - 私聊测试Agent功能
   - 小群试运行
   - 再部署到正式群

2. **权限最小化**
   - 只给必要的工具权限
   - 避免误操作风险

3. **兜底机制**
   - 设置人工介入通道
   - 复杂问题转人工

4. **监控日志**
   - 记录Agent回复内容
   - 定期review效果
