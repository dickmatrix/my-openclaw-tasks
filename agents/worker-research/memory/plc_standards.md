# 西门子 PLC 竞品技术手册摘要

## 西门子 S7 系列

### S7-1200
- **定位**：小型 PLC，入门级自动化控制
- **CPU**：1214C DC/DC/DC（14输入/10输出）
- **通信**：PROFINET、RS485
- **编程**：TIA Portal V17+
- **优势**：运动控制集成、Modbus TCP

### S7-1500
- **定位**：中大型 PLC，旗舰级控制系统
- **性能**：位指令 0.08μs，高速运动控制
- **安全**：内置安全功能 (F-CPU)
- **诊断**：Profinet 诊断、Web 服务器
- **扩展**：最多 32 个模块

## 三菱电机 iQ-R 系列

### iQ-R CPU
- **处理器**：双核 64 位 RISC
- **执行速度**：0.98ns/step
- **程序容量**：2M 步
- **通信**：CC-Link IE、NestJS 兼容
- **冗余**：支持双机热备

### GX Works3
- **编程语言**：ST、FBD、LAD、SFC
- **仿真**：内置仿真器
- **调试**：在线监控、断点调试

## 欧姆龙 NX/NJ 系列

### NX1P2
- **定位**：一体化 CPU + 运动控制
- **轴数**：最多 4 轴内置轴
- **EtherCAT**：原生支持高速运动
- **安全**：安全功能块认证

### Sysmac Studio
- **统一平台**：PLC + 运动 + 视觉
- **配置**：数据库连接 (SQL)
- **仿真**：虚拟控制器

## 关键参数对比

| 品牌 | 系列 | 响应时间 | 程序容量 | 价格区间 |
|------|------|----------|----------|----------|
| 西门子 | S7-1500 | 0.08μs | 8MB | 中高端 |
| 三菱 | iQ-R | 0.98ns | 2M步 | 高端 |
| 欧姆龙 | NX1P2 | 0.4μs | 5MB | 中高端 |

## 选型建议
1. **小型项目**：优先 S7-1200 或 NX1P2
2. **中大型项目**：S7-1500 或 iQ-R
3. **运动控制密集**：欧姆龙 EtherCAT 原生支持优势
4. **安全系统**：西门子 F-CPU 认证完善
