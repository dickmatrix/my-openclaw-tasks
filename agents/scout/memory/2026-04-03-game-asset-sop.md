## 三、执行 SOP（标准化流程）

### Step 1：选方向 & 定 Niche

**规则：细分 > 泛化**
- ❌ 不要做"通用游戏图标100个"
- ✅ 做"像素风 RPG 道具图标 80个（64x64）"

**选品公式：**
```
[游戏类型] + [美术风格] + [资产类型] + [规格参数]
例如：像素风 / RPG / 武器图标 / 32x32 / 水墨风 / 卡通风
```

**每日选品决策：**
1. 打开 Unity Asset Store 搜索关键词，看同类产品的评价数（>50 = 有市场）
2. 打开 Itch.io 热销榜，看哪些标签频繁出现
3. 选供< 需的细分方向

---

### Step 2：批量生成（算力驱动）

**工具推荐：**
- 图像生成：**Stable Diffusion**（本地/云端）或 **Midjourney**
- 修图：**Photoshop + AI插件**（创可贴）/ **Clipdrop**
- 批量处理：**Python + PIL** 批量裁剪/重命名

**生成参数模板（SD）：**
```
Steps: 30
CFG Scale: 7-9
Size: 512x512（图标）、1024x1024（立绘）
风格关键词：pixel art, 16-bit, RPG, game asset, transparent background
否定词：blurry, low quality, watermark, text, logo
```

**批量生成策略：**
- 每个品类一次生成 20-50 张
- 人工筛选保留 5-15 张优质结果
- 修图师精修（外包，5-20元/张）

---

### Step 3：整理 & 打包

**标准化命名：**
```
[类型]_[风格]_[编号].png
例如：icon_sword_001.png, icon_potion_002.png
```

**标准化规格：**
- 图标：32x32 / 64x64 / 128x128（多规格打包）
- 贴图：512x512 / 1024x1024（PBR贴图含 Albedo/Normal/Roughness）
- 立绘：1024x1024 / 2048x2048（PNG透明背景）

**打包格式：**
- Unity Asset Store：`.unitypackage` 或直接上传资源包
- Itch.io：`.zip` 压缩包
- 价格区间：$5-$30（图标包）/ $10-$50（材质包）/ $5-$20（BGM包）

---

### Step 4：发布 & 上架

**Itch.io 上架 Checklist：**
- [ ] 封面图（吸引眼球，1280x720）
- [ ] 预览图（3-6张展示不同资产）
- [ ] 描述文案（包含关键词：tags）
- [ ] 价格设定（建议 $5-$15 起）
- [ ] 关联标签（pixel art, RPG, icons, game assets...）

**Unity Asset Store 上架 Checklist：**
- [ ] 准备商家主页
- [ ] 上传资源包（<500MB）
- [ ] 填写资产描述、标签、截图
- [ ] 设定价格（建议 $10-$30）
- [ ] 提交审核（3-7天）

---

### Step 5：变现 & 放大

**初期策略：**
- 低价快速出单（$5 试水）→ 积累评价
- 有 5+ 评价后适当提价
- 定期上新（每周 1-2 个新包）

**放大策略：**
- 同一资产类型做系列（RPG图标 V1/V2/V3...）
- 跨平台分发（Itch.io + Unity + Unreal）
- Patreon 订阅解锁更快变现
