# Zed 必装扩展清单（通用、低维护）

luz 于 2026-04-04 10:17 发送

## 必装扩展

| # | 扩展名 | 作用 |
|---|--------|------|
| 1 | GitHub MCP Server | 让 Agent 直接读仓库、Issue、PR，减少手动切网页 |
| 2 | Context7 MCP Server | 让 Agent 查官方文档（比普通搜索更稳） |
| 3 | Brave Search MCP Server | 做外部检索兜底（当文档/仓库里找不到答案时） |
| 4 | Prettier | 一键格式化，减少"为什么这段格式不一致" |
| 5 | ESLint | 有 TS/JS 就必装，保存即提示问题，减少低级错误 |
| 6 | Material Icon Theme | 解决日志报错 `icon theme not found: Material Icon Theme` |
| 7 | Catppuccin Theme（含 Mocha） | 解决日志报错 `theme not found: Catppuccin Mocha` |
