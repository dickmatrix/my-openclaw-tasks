# Zed 编辑器小白零基础操作教程

> 来源：用户分享 | 日期：2026-04-03 | 格式：docx 内容整理

## 核心定位

Zed 是基于 Rust 开发的高性能代码编辑器，主打 GPU 加速、原生 AI 集成和实时协作。

## 一、安装与首次启动配置

### 1. 安装
直接前往 [Zed 官网](https://zed.dev/) 下载对应系统（Windows/macOS/Linux）的安装包，全程默认下一步即可完成安装，无捆绑插件。

### 2. 首次启动核心配置
- **快捷键映射选择**：启动后可直接选择 VS Code/IntelliJ IDEA/Sublime 等熟悉的编辑器快捷键方案；Vim 用户可单独开启 Vim Mode
- **配置导入**：支持一键导入 VS Code/Cursor 的基础配置
- **主题选择**：内置多款明暗双模式主题，支持跟随系统自动切换

## 四、重点细节&避坑指南

### 核心功能细节
- **命令面板隐藏用法**：输入 @ 跳转到函数，输入 # 全项目搜符号，输入 : 跳转到指定行
- **智能选中**：Alt+↑/↓ 按语法层级扩大/缩小选中范围
- **多光标编辑**：Cmd/Ctrl+点击 添加多个光标
- **Git 集成**：行号旁颜色标记（绿=新增，蓝=修改，红=删除）
- **AI 功能**：选中代码按 Cmd+L 唤起 AI 对话

### 避坑指南
- **启动失败**：检查显卡驱动，更新 Vulkan；Linux 需 GLIBC 升级
- **启动闪退**：删除配置文件夹（macOS: ~/.config/zed/）重装
- **Vim 模式不生效**：需在设置中手动开启
- **代码无提示**：检查对应语言运行环境是否安装（LSP 依赖环境）
- **性能优化**：避免过多扩展；超大项目配置排除文件夹（node_modules、venv）

## 五、5分钟上手步骤

1. 打开 Zed → 「打开文件夹」→ 选择项目
2. Cmd+N 新建文件（如 test.py）
3. `Ctrl+`` 打开内置终端
4. 故意制造语法错误 → 按 F8 跳转查看
5. Cmd+\ 拆分编辑区（左侧写代码、右侧看文档）
6. Cmd+Shift+P → 搜索「主题」→ 切换主题
7. Cmd+P → 输入文件名 → 快速跳转
