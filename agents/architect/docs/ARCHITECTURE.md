# Quant_Analyst 架构设计

## 核心目标
- **趋势导向** > 实时行情
- **方向性指导** > 模糊判断
- **置信度加权** > 平均主义
- **板块信号** > 个股噪音

## 数据层
- 美股: 3年历史数据 (YFinance)
- 港股: 趋势指标为主，实时行情为辅
- 宏观: 月度频率 (CPI/FEDFUNDS/M2/GOLD/OIL)

## 方法论置信度体系
| 方法论 | 置信度 | 权重因子 |
|--------|--------|----------|
| 趋势跟踪 (MA200) | ★★★★★ | 0.25 |
| 12M 动量 | ★★★★☆ | 0.20 |
| Fama-French 三因子 | ★★★★☆ | 0.15 |
| 风险平价 (Dalio) | ★★★★☆ | 0.15 |
| 价值投资 (PB/ROIC) | ★★★☆☆ | 0.10 |
| RSI 均值回归 | ★★★☆☆ | 0.08 |
| 美林时钟 | ★★★☆☆ | 0.07 |

## 信号评分公式
```
板块评分 = Σ(方法论权重 × 信号强度 × 置信度系数)
信号强度: -1 (强烈卖出) / 0 (中性) / +1 (强烈买入)
置信度系数: 0.5 ~ 1.0 (根据历史胜率调整)
```

## 新闻情感权重
- 财新/东方财富关键词情感打分
- 权重占比: 总信号的 15%
- 情感极端时权重放大 (超买/超卖新闻)

## 日报结构 (专家级)
1. **宏观环境定调** - 无须细节，给方向
2. **板块信号矩阵** - 买入/持有/卖出
3. **仓位建议** - 根据置信度
4. **风险提示** - 最大威胁
5. **新闻情感摘要** - 辅助判断

## 知识库压缩节奏
- 每周: 宏观异动因子归纳
- 每月: 策略胜率校准
- 每季度: 向量合并归档
